    1. Initialize root package.json with `"private": true`, name "tether", and devDependencies: turbo, typescript, eslint, prettier, vitest. Scripts: "build": "turbo build", "dev": "turbo dev", "lint": "turbo lint", "test": "turbo test", "typecheck": "turbo typecheck". Set "packageManager": "pnpm@9.15.4" (or current pnpm version).

    2. Create pnpm-workspace.yaml:
       ```yaml
       packages:
         - "apps/*"
         - "packages/*"
       ```

    3. Create turbo.json (Turborepo 2.x format):
       ```json
       {
         "$schema": "https://turbo.build/schema.json",
         "tasks": {
           "build": { "dependsOn": ["^build"], "outputs": ["dist/**"] },
           "dev": { "cache": false, "persistent": true },
           "lint": { "dependsOn": ["^lint"] },
           "test": { "dependsOn": ["build"] },
           "typecheck": { "dependsOn": ["^build"] }
         }
       }
       ```

    4. Create tsconfig.base.json with strict TypeScript settings: strict true, ESNext target, module "ESNext", moduleResolution "bundler", esModuleInterop, skipLibCheck, forceConsistentCasingInFileNames, resolveJsonModule. This is the base config all packages extend.

    5. Create packages/shared:
       - package.json: name "@tether/shared", private true, type "module", exports: { ".": { "types": "./src/index.ts", "import": "./src/index.ts" } }. Scripts: "typecheck": "tsc --noEmit", "lint": "eslint src/".
       - tsconfig.json: extends ../../tsconfig.base.json, include ["src"], compilerOptions: { outDir "dist", rootDir "src" }.
       - src/index.ts: export a placeholder `export const TETHER_VERSION = "0.1.0";` and re-export from types (empty for now).

    6. Create .gitignore with: node_modules, dist, .turbo, .env, *.local, .DS_Store.

    7. Create .prettierrc with sensible defaults (semi: true, singleQuote: false, trailingComma: "all", printWidth: 100, tabWidth: 2).

    8. Create .prettierignore with: node_modules, dist, .turbo, pnpm-lock.yaml.

    9. Create eslint.config.js using flat config format (ESLint 9.x). Include typescript-eslint plugin. Keep it minimal — just TypeScript recommended rules.
  </action>
  <verify>
    Run `pnpm install` from root — no errors. Run `pnpm typecheck --filter @tether/shared` — succeeds.
  </verify>
  <done>Root monorepo config exists, shared package compiles, pnpm workspace resolves.</done>
</task>

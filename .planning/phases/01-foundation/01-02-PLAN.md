<task type="auto">
  <name>Task 1: Drizzle schema with all tables and crypto columns</name>
  <files>
    apps/server/package.json
    apps/server/src/db/schema.ts
    apps/server/src/db/client.ts
    apps/server/drizzle.config.ts
  </files>
  <action>
    1. Install Drizzle dependencies in apps/server:
       `pnpm add drizzle-orm postgres` and `pnpm add -D drizzle-kit`

    2. Create apps/server/src/db/schema.ts with ALL tables for the entire project. Use uuid primary keys with defaultRandom(). Use Drizzle's pgTable, uuid, text, bytea, timestamp, integer, boolean, smallint from "drizzle-orm/pg-core". If `bytea` is not directly importable, use `customType` to define it (see RESEARCH.md Pitfall 4).

    Tables to define:

    **users** — Core auth + crypto storage:
    - id: uuid PK defaultRandom
    - email: text notNull unique
    - displayName: text notNull
    - authKeyHash: text("auth_key_hash") notNull — Argon2id hash of auth key
    - x25519PublicKey: bytea("x25519_public_key") notNull
    - ed25519PublicKey: bytea("ed25519_public_key") notNull
    - x25519EncryptedPrivateKey: bytea("x25519_encrypted_private_key") notNull
    - ed25519EncryptedPrivateKey: bytea("ed25519_encrypted_private_key") notNull
    - x25519KeyIv: bytea("x25519_key_iv") notNull — AES-GCM nonce for x25519 private key
    - ed25519KeyIv: bytea("ed25519_key_iv") notNull — AES-GCM nonce for ed25519 private key
    - kdfSalt: bytea("kdf_salt") notNull — PBKDF2 salt, 32 bytes
    - recoveryKeyHash: text("recovery_key_hash") — hash of one-time recovery key
    - avatarUrl: text("avatar_url")
    - status: text("status").default("online")
    - createdAt: timestamp("created_at").defaultNow().notNull()
    - updatedAt: timestamp("updated_at").defaultNow().notNull()

    **refresh_tokens** — JWT refresh token rotation:
    - id: uuid PK defaultRandom
    - userId: uuid("user_id") notNull references users.id (onDelete cascade)
    - jti: text("jti") notNull unique — JWT ID for rotation detection
    - expiresAt: timestamp("expires_at") notNull
    - createdAt: timestamp("created_at").defaultNow().notNull()

    **servers** — Discord-like guilds:
    - id: uuid PK defaultRandom
    - name: text("name") notNull
    - ownerId: uuid("owner_id") notNull references users.id
    - iconUrl: text("icon_url")
    - createdAt: timestamp("created_at").defaultNow().notNull()
    - updatedAt: timestamp("updated_at").defaultNow().notNull()

    **channels** — Text and voice channels within servers:
    - id: uuid PK defaultRandom
    - serverId: uuid("server_id") notNull references servers.id (onDelete cascade)
    - name: text("name") notNull
    - type: text("type").notNull().default("text") — "text" | "voice" | "dm"
    - topic: text("topic")
    - position: integer("position").notNull().default(0)
    - createdAt: timestamp("created_at").defaultNow().notNull()
    - updatedAt: timestamp("updated_at").defaultNow().notNull()

    **messages** — Encrypted message storage (ciphertext only):
    - id: uuid PK defaultRandom
    - channelId: uuid("channel_id") notNull references channels.id (onDelete cascade)
    - senderId: uuid("sender_id") notNull references users.id
    - encryptedContent: bytea("encrypted_content") notNull — AES-256-GCM ciphertext
    - contentIv: bytea("content_iv") notNull — nonce for this message
    - contentAlgorithm: text("content_algorithm").notNull().default("aes-256-gcm")
    - epoch: integer("epoch").notNull().default(1) — key rotation epoch
    - createdAt: timestamp("created_at").defaultNow().notNull()
    - editedAt: timestamp("edited_at")

    **message_recipient_keys** — Per-recipient wrapped message key:
    - id: uuid PK defaultRandom
    - messageId: uuid("message_id") notNull references messages.id (onDelete cascade)
    - recipientUserId: uuid("recipient_user_id") notNull references users.id
    - encryptedMessageKey: bytea("encrypted_message_key") notNull — message AES key wrapped with recipient's X25519 public key
    - ephemeralPublicKey: bytea("ephemeral_public_key") notNull — sender's ephemeral X25519 public key for this wrap
    - Add unique constraint on (messageId, recipientUserId)

    **server_members** — Membership join table:
    - id: uuid PK defaultRandom
    - serverId: uuid("server_id") notNull references servers.id (onDelete cascade)
    - userId: uuid("user_id") notNull references users.id (onDelete cascade)
    - joinedAt: timestamp("joined_at").defaultNow().notNull()
    - Add unique constraint on (serverId, userId)

    **roles** — Server-scoped roles with permission bitfields:
    - id: uuid PK defaultRandom
    - serverId: uuid("server_id") notNull references servers.id (onDelete cascade)
    - name: text("name") notNull
    - permissions: text("permissions").notNull().default("0") — bigint as string for bitfield
    - color: text("color")
    - position: integer("position").notNull().default(0)
    - createdAt: timestamp("created_at").defaultNow().notNull()

    **member_roles** — Many-to-many members-to-roles:
    - memberId: uuid("member_id") notNull references server_members.id (onDelete cascade)
    - roleId: uuid("role_id") notNull references roles.id (onDelete cascade)
    - Primary key on (memberId, roleId)

    **invites** — Server invite codes:
    - id: uuid PK defaultRandom
    - serverId: uuid("server_id") notNull references servers.id (onDelete cascade)
    - creatorId: uuid("creator_id") notNull references users.id
    - code: text("code") notNull unique
    - maxUses: integer("max_uses")
    - uses: integer("uses").notNull().default(0)
    - expiresAt: timestamp("expires_at")
    - createdAt: timestamp("created_at").defaultNow().notNull()

    **channel_overrides** — Per-channel role permission overrides:
    - id: uuid PK defaultRandom
    - channelId: uuid("channel_id") notNull references channels.id (onDelete cascade)
    - roleId: uuid("role_id") notNull references roles.id (onDelete cascade)
    - allow: text("allow").notNull().default("0") — bitfield of allowed perms
    - deny: text("deny").notNull().default("0") — bitfield of denied perms
    - Add unique constraint on (channelId, roleId)

    Export all tables and their inferred types (InferSelectModel, InferInsertModel) for use in route handlers.

    3. Create apps/server/src/db/client.ts:
       - Import postgres from "postgres" and drizzle from "drizzle-orm/postgres-js"
       - Import all schema tables with `import * as schema from "./schema.js"`
       - Create connection: `const queryClient = postgres(process.env.DATABASE_URL!)`
       - Export: `export const db = drizzle(queryClient, { schema })`
       - Note: Use .js extension in import for ESM compatibility

    4. Create apps/server/drizzle.config.ts:
       ```typescript
       import { defineConfig } from "drizzle-kit";
       export default defineConfig({
         schema: "./src/db/schema.ts",
         out: "./src/db/migrations",
         dialect: "postgresql",
         dbCredentials: { url: process.env.DATABASE_URL! },
       });
       ```

    5. Add "db:generate": "drizzle-kit generate", "db:migrate": "drizzle-kit migrate", "db:push": "drizzle-kit push" to apps/server/package.json scripts.
  </action>

    1. Create packages/shared/src/types/crypto.ts with types used across client and server:
       ```typescript
       // Encrypted private key blob as stored/transmitted
       export interface EncryptedKeyBlob {
         ciphertext: Uint8Array;  // AES-256-GCM encrypted PKCS8 key
         iv: Uint8Array;          // 12-byte nonce
       }

       // Full key bundle returned to client on login
       export interface KeyBundle {
         x25519PublicKey: Uint8Array;
         ed25519PublicKey: Uint8Array;
         x25519EncryptedPrivateKey: EncryptedKeyBlob;
         ed25519EncryptedPrivateKey: EncryptedKeyBlob;
         kdfSalt: Uint8Array;
       }

       // Key derivation parameters
       export interface KdfParams {
         algorithm: "PBKDF2-SHA256";
         iterations: number;
         saltLength: number;
         hkdfInfo: {
           auth: string;      // e.g., "tether-auth-key-v1"
           encryption: string; // e.g., "tether-encryption-key-v1"
         };
       }

       // Constants
       export const KDF_ITERATIONS = 600_000;
       export const KDF_SALT_LENGTH = 32;
       export const AES_GCM_IV_LENGTH = 12;
       export const AUTH_HKDF_INFO = "tether-auth-key-v1";
       export const ENCRYPTION_HKDF_INFO = "tether-encryption-key-v1";
       ```

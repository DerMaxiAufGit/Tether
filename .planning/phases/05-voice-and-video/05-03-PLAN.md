Create `apps/client/src/hooks/useVoiceChannel.ts` — the central hook that owns all WebRTC state.

**State shape:**
```typescript
interface VoiceState {
  channelId: string | null;
  serverId: string | null;
  connectionState: "idle" | "requesting-mic" | "joining" | "connected" | "failed";
  participants: VoiceParticipant[];
  localStream: MediaStream | null;
  remoteStreams: Map<string, MediaStream>; // userId -> MediaStream
  muted: boolean;
  deafened: boolean;
  cameraOn: boolean;
  screenShares: Map<string, MediaStream>; // streamId -> MediaStream
  speaking: boolean;
  error: string | null;
}
```

**Storage:**
- `peersRef = useRef<Map<string, RTCPeerConnection>>(new Map())` — mutable PC map (anti-pattern: never in useState)
- `pendingCandidatesRef = useRef<Map<string, RTCIceCandidateInit[]>>(new Map())` — ICE candidate buffer per peer
- `localStreamRef = useRef<MediaStream | null>(null)` — local mic/camera stream
- `makingOfferRef = useRef<Map<string, boolean>>(new Map())` — per-peer flag for perfect negotiation

**Core flow — `join(channelId, serverId)`:**
1. Set state to `"requesting-mic"`
2. Call `navigator.mediaDevices.getUserMedia({ audio: true })` — if denied, set error and return (per locked decision: mic required, block join until granted)
3. Store stream in `localStreamRef`
4. Set state to `"joining"`
5. Fetch TURN credentials: `api.get<TurnCredentialsResponse>("/api/voice/turn-credentials")`
6. Store `iceServers` in ref
7. Emit `socket.emit("voice:join", { channelId })`
8. Wait for `voice:joined` event (handler below creates connections)

**Socket event handlers (register in useEffect, clean up on unmount):**
Use stable wrapper refs (pattern from 03-03, 04-05) for all async handlers.

- **voice:joined** `(VoiceJoinedPayload)`:
  - Set participants state
  - Set connectionState to `"connected"`
  - For each existing participant, call `createPeerConnection(participant.userId)` — the joiner initiates offers to all existing peers

- **voice:participant_joined** `(VoiceParticipantJoinedPayload)`:
  - Add to participants state
  - Call `createPeerConnection(participant.userId)` — existing participant creates PC for the new peer (but does NOT create offer; the new joiner sends offers)

- **voice:participant_left** `(VoiceParticipantLeftPayload)`:
  - Close and delete `peersRef.current.get(userId)`
  - Remove from participants state
  - Remove from remoteStreams

- **voice:offer** `({ from, sdp, signature })`:
  - Call `handleDescription(from, sdp)` (perfect negotiation)

- **voice:answer** `({ from, sdp, signature })`:
  - Call `handleDescription(from, sdp)` (perfect negotiation)

- **voice:ice** `({ from, candidate })`:
  - If remote description not yet set for this peer, buffer in `pendingCandidatesRef`
  - Otherwise call `peersRef.current.get(from)?.addIceCandidate(candidate)`

- **voice:mute/deafen/camera/speaking/screen_share**: update the matching participant in state

**createPeerConnection(peerId):**
```typescript
function createPeerConnection(peerId: string) {
  const pc = new RTCPeerConnection({
    iceServers: iceServersRef.current,
    iceTransportPolicy: "relay", // TURN only — prevents local IP exposure (per research)
    bundlePolicy: "max-bundle",
  });

  // Add local audio track to the connection
  if (localStreamRef.current) {
    for (const track of localStreamRef.current.getTracks()) {
      pc.addTrack(track, localStreamRef.current);
    }
  }

  // ICE candidate handler — relay to remote peer via signaling
  pc.onicecandidate = ({ candidate }) => {
    if (candidate) {
      socket.emit("voice:ice", { to: peerId, candidate: candidate.toJSON() });
    }
  };

  // Remote track handler — store remote stream
  pc.ontrack = (event) => {
    const [stream] = event.streams;
    if (stream) {
      setRemoteStreams(prev => new Map(prev).set(peerId, stream));
    }
  };

  // Perfect negotiation: onnegotiationneeded
  pc.onnegotiationneeded = async () => {
    try {
      makingOfferRef.current.set(peerId, true);
      await pc.setLocalDescription(); // implicit offer
      socket.emit("voice:offer", {
        to: peerId,
        sdp: pc.localDescription!,
        signature: "" // SDP signing added later
      });
    } catch (err) {
      console.error("Negotiation error:", err);
    } finally {
      makingOfferRef.current.set(peerId, false);
    }
  };

  // ICE connection state monitoring
  pc.oniceconnectionstatechange = () => {
    if (pc.iceConnectionState === "failed") {
      pc.restartIce(); // triggers negotiationneeded → new offer
    }
  };

  peersRef.current.set(peerId, pc);
  return pc;
}
```

**handleDescription(peerId, description) — Perfect Negotiation:**
```typescript
async function handleDescription(peerId: string, description: RTCSessionDescriptionInit) {
  const pc = peersRef.current.get(peerId);
  if (!pc) return;

  const polite = userId < peerId; // lexicographic comparison
  const offerCollision = description.type === "offer" &&
    (makingOfferRef.current.get(peerId) || pc.signalingState !== "stable");
  const ignoreOffer = !polite && offerCollision;
  if (ignoreOffer) return;

  await pc.setRemoteDescription(description);

  // Flush buffered ICE candidates
  const pending = pendingCandidatesRef.current.get(peerId) ?? [];
  for (const candidate of pending) {
    await pc.addIceCandidate(candidate);
  }
  pendingCandidatesRef.current.delete(peerId);

  if (description.type === "offer") {
    await pc.setLocalDescription(); // implicit answer
    socket.emit("voice:answer", {
      to: peerId,
      sdp: pc.localDescription!,
      signature: ""
    });
  }
}
```

**leave():**
1. Emit `socket.emit("voice:leave", { channelId })`
2. Close all peer connections in `peersRef`
3. Stop all local stream tracks
4. Clear all state back to idle

**toggleMute():**
- Toggle `localStreamRef.current.getAudioTracks()[0].enabled`
- Update local muted state
- Emit `socket.emit("voice:mute", { channelId, muted: newValue })`

**toggleDeafen():**
- Set all remote audio tracks `.enabled = !deafened`
- Update local deafened state
- Emit `socket.emit("voice:deafen", { channelId, deafened: newValue })`
- If deafening, also mute (convention: deafen implies mute)

**Cleanup on unmount / channelId change:**
- Close all PCs, stop local stream, clear refs
- Remove socket event listeners using stable wrapper refs

import { createContext, useContext, type ReactNode } from "react";
import { useVoiceChannel } from "../hooks/useVoiceChannel";

// The return type of useVoiceChannel
type VoiceContextType = ReturnType<typeof useVoiceChannel>;

const VoiceContext = createContext<VoiceContextType | null>(null);

export function VoiceProvider({ children }: { children: ReactNode }) {
  const voice = useVoiceChannel();
  return <VoiceContext.Provider value={voice}>{children}</VoiceContext.Provider>;
}

export function useVoice(): VoiceContextType {
  const ctx = useContext(VoiceContext);
  if (!ctx) throw new Error("useVoice must be used within VoiceProvider");
  return ctx;
}
